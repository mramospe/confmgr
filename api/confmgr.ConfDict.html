ConfDict¶

class confmgr.ConfDict(*args, **kwargs)[source]¶

Bases: dict, confmgr.core.ConfObj

Represent a dictionary to store configurations. Some methods of the dict class are overriden. The classes must be stored using the Config class, unless they can be built using their string representation.
