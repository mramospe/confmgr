ConfMgr¶

class confmgr.ConfMgr(*args, **kwargs)[source]¶

Bases: confmgr.core.ConfDict

Class to manage configurations built using the Config class. It allows to read/save the stored information from/to XML files. This class is constructed as a configuration dictionary.
